# Secret Code Language Generator (Python)

- A fun program that converts normal text into a secret coded language
- Users can encode and decode messages
- Uses custom rules to transform words
- Helps understand string manipulation in Python
- Can be used for basic encryption practice

# 📚 Topics Covered
- Strings & String Slicing
- User Input (input())
- Conditional Statements (if-else)
- Loops
- Random Module (random) (if used)
- Functions (if you created encode/decode functions)
- Logic Building
